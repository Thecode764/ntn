<h2 align="center">Install command</h2>
<p align="center">Use this command for install npm packages</p>
<p align="center">

```
ntn --i package-name
```    

</p>
OR
<p align="center">

```
ntn --in package-name
```    

</p>
OR
<p align="center">

```
ntn --ins package-name
```    

</p>
OR
<p align="center">

```
ntn --inst package-name
```    

</p>
OR
<p align="center">

```
ntn --insta package-name
```    

</p>
OR
<p align="center">

```
ntn --instal package-name
```    

</p>
OR
<p align="center">

```
ntn --install package-name
```
